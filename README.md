## Paquete ull-Triangulo-lucas-ivan-miguel-jairo

Primer paso:

```
npm install ull-Triangulo-lucas-ivan-miguel-jairo
```
Segundo paso, incluír el paquete en tu programa:

```javascript
require('ull-Triangulo-lucas-ivan-miguel-jairo')
```
